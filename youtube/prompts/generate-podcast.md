# NotebookLM Research Paper Podcast Prompt

Generate a podcast that covers:

## Core Content

- Main thesis/hypothesis and why it matters
- Key methodology and novel approaches
- Most significant findings with concrete examples
- Surprising/counterintuitive results
- Practical implications and applications

## Context

- How this builds on/differs from prior work
- What problem does this solve
- Limitations and caveats
- Open questions raised

## Format

- Always start with greeting: "Witamy ponownie w głębokim odczycie"
- Follow with why this paper matters (hook)
- Use examples and analogies for complex concepts
- Compare/contrast with existing approaches
- End with "so what" - impact and next steps

## Language

- Keep ALL technical terms in English (no translations)
- Preserve original names: algorithms, architectures, frameworks, libraries, protocols, patterns
- Examples: "transformer", "attention mechanism", "fine-tuning", "embedding" - never translate

## Avoid

- Jargon without explanation
- Exhaustive methodology details
- Every citation
- Minor/incremental findings

Focus on insights someone would need to understand and discuss this paper intelligently.

---

## Usage

Paste as instructions when uploading paper to NotebookLM before generating podcast.

## Customizations

- Add `"Focus on [specific aspect]"` for targeted coverage
- Add `"Compare with [other paper]"` for comparative analysis
- Add `"Explain for [audience level]"` for specific depth
